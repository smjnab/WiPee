# WiPee
Micro Python IoT Water Monitor Using a WiPy

## Details
Run code on a WiPy and with two cables connected to port 11 and 12 the WiPy will notice 
connection and trigger an email using AWS services. 

## Dependencies
* https://pycom.io/hardware/wipy-3-0-specs/ with latest firmware.
* Pycoms AWSIoTPythonSDK, https://github.com/pycom/aws-pycom/tree/master/AWSIoTPythonSDK
* AWS account https://aws.amazon.com to use IoT, Lambda and SES.
